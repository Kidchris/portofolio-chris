N8N + LLM + RAG sur Kubernetes

GOLUMINA

Automatisation IA avec N8N, LLM et RAG sur Kubernetes

Mission DevOps & IA en cours depuis septembre 2025 pour Golumina, en remote. Mise en place d'une plateforme d'automatisation intelligente basée sur N8N, des LLMs open source et une architecture RAG (Retrieval-Augmented Generation) déployée sur Kubernetes.

Pipeline d'automatisation IA

Déploiement N8N sur Docker/Kubernetes, configuration de pipelines d'automatisation intégrant des LLMs open source pour le traitement du langage naturel. Mise en place de bases vectorielles (Chroma/Weaviate) pour la recherche sémantique et le RAG.

Infrastructure CI/CD

Pipeline CI/CD GitLab, NGINX reverse proxy, backend FastAPI (Python) pour les endpoints IA. Architecture scalable sur Kubernetes permettant une montée en charge automatique des services.
